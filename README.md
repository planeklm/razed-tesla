# How To Install
* Drag and drop `razed-tesla`
* In your `server.cfg` add `ensure razed-tesla`
* In  `client.lua` change LegacyFuel to your fuel, or change its export.


# Config
```
Config = {}

Config.Tesla = -429774847 -- Your Tesla hashkey.
```
